# Private Scalvin Workspace

This directory contains a private, local Scalvin workspace. It may include
personal reflections, session summaries, source documents, transcripts, and
other sensitive material.

The workspace uses a default-deny `.gitignore`. Do not publish it or remove
that protection unless the owner has made an informed decision about every
file that would become trackable.

Use the installed Scalvin commands to inspect, export, back up, restore,
correct, forget, or delete data. Framework source and public contributions
belong in the separate Scalvin source repository, never in this workspace.
