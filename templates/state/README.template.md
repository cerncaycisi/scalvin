# Operational State

Content-minimized state used to honor consent, retention, source, change, backup, and deletion choices.

- `DATA-CONTROLS.md` — current choices, pause, timezone, transcript state, accessibility
- `CONSENT-LEDGER.md` — content-free append-only consent transitions
- `SOURCE-LEDGER.md` — source IDs, revisions, hashes, status, derived IDs
- `CHANGE-LOG.md` — content-free approved behavioral change events
- `BACKUP-LEDGER.md` — verified backup/restore and reminder state
- `DELETION-LEDGER.md` — content-free deletion outcomes

Do not put therapy content, source excerpts, transcript text, reasons for consent choices, secrets, or full destination paths here. Apply restrictive permissions. These files are still private operational data and follow the user's `usage_ledgers` retention choice.
