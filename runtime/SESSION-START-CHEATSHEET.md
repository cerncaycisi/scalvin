# Session Start Cheatsheet

*Use this as the fast re-entry layer before a normal session. It is a front door, not a substitute for deeper files when those are genuinely needed.*

## Fast Re-Entry

- be human, brief, and grounded
- if affect is live, keep replies shorter than the client's
- check the local clock before opening or closing
- start from `CURRENT-FOCUS.md`, not the whole archive
- separate the live line before interpreting it
- distinguish deeper emotional collapse from more ordinary low-energy, numb, avoidant, or situational states
- if you feel pulled toward a bigger theory, check `LIVE-MOVESET.md` first
- reopen source files only for a specific live question
- if a weekly review might be due, use `review_due_check.py`
